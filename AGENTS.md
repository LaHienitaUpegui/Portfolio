# Portfolio Project

An Astro-based personal portfolio website with React components and GSAP animations.

## Tech Stack

- **Framework**: Astro 5.17.1
- **UI**: React 19.2.4
- **Animations**: GSAP 3.14.2
- **Styling**: CSS with CSS variables

## Commands

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run astro` - Run Astro CLI

## Project Structure

- `src/pages/` - Astro pages (index, portfolio-content, test)
- `src/components/` - React components (AboutMe, Projects, ProjectCard, MyExperience, ExperienceCard, MoreComingSoon, MainContentComponent)
- `src/layouts/` - Astro layouts
- `src/styles/` - CSS files
- `src/assets/` - Images and SVGs
- `public/fonts/` - Font files

## Key Files

- `src/pages/index.astro` - Landing page with interactive button
- `src/pages/portfolio-content.astro` - Main portfolio content
- `src/layouts/Layout.astro` - Base layout
